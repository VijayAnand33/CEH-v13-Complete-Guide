## Overview

Anti DDoS Guardian is a real-time network protection solution designed to detect, analyze, and mitigate Denial-of-Service (DoS) and Distributed Denial-of-Service (DDoS) attacks. It continuously monitors incoming and outgoing network traffic, identifies abnormal connection patterns, and allows administrators to block malicious IP addresses to maintain service availability.

---

## Key Features

- Real-time traffic monitoring
- DoS and DDoS attack detection
- IP address blocking
- Traffic session analysis
- TCP and UDP traffic monitoring
- Connection rate limiting
- Network flow inspection
- Attack mitigation

---

## Common Usage

Typical workflow:

1. Launch Anti DDoS Guardian.
2. Monitor incoming network traffic.
3. Identify suspicious connections.
4. Inspect traffic details.
5. Block malicious IP addresses.

## CEH Practical Example

During **Module 10 – Denial-of-Service**, Anti DDoS Guardian was used to monitor network traffic generated by multiple LOIC attackers. The tool identified abnormal traffic patterns, displayed detailed session information, and mitigated the attack by blocking the malicious IP addresses responsible for the DDoS traffic.
