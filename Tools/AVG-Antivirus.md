# AVG Antivirus

## Overview

AVG Antivirus is a mobile security solution that protects Android devices from malware, spyware, ransomware, and other mobile threats. It performs security assessments by scanning installed applications and device storage to detect and remove malicious software.

---

## Key Features

- Detects malware and malicious applications.
- Performs full device security scans.
- Removes detected threats.
- Monitors application security.
- Protects against common Android malware.
- Improves overall mobile device security.

---

## Common Usage

- Launch AVG Antivirus
- Perform Security Scan
- Review Detected Threats
- Remove Malware

---

## Practical Example

During **Module 17 – Hacking Mobile Platforms**, AVG Antivirus was used to perform a security assessment of an Android device by scanning for malicious applications, detecting malware, and removing identified threats to improve device security.

---

## Defensive Perspective

Regular antivirus scans, timely malware removal, operating system updates, and careful application installation practices help protect Android devices from evolving mobile threats and unauthorized access.
