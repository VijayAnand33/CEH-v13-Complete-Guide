# Resource Monitor

> **Category:** System Performance Monitoring Tool
>
> **Platform:** Windows
>
> **License:** Built-in
>
> **Official Website:** https://learn.microsoft.com/windows/

---

## Overview

Resource Monitor is a built-in Windows performance monitoring utility that provides detailed, real-time information about CPU, memory, disk, and network resource utilization. It helps administrators and security professionals identify resource bottlenecks, analyze system behavior, and investigate the impact of security incidents such as DoS attacks.

---

## Key Features

- CPU utilization monitoring
- Memory usage analysis
- Disk activity monitoring
- Network utilization monitoring
- Process performance analysis
- Real-time resource statistics
- Service monitoring
- Performance troubleshooting

---

## Common Usage

Launch Resource Monitor:

```text
resmon
```

Common workflow:

1. Launch Resource Monitor.
2. Select the desired performance tab.
3. Observe system resource utilization.
4. Identify abnormal activity.
5. Investigate resource-intensive processes.

---

## CEH Practical Example

During **Module 10 – Denial-of-Service**, Resource Monitor was used to observe the impact of TCP flooding attacks generated by ISB and UltraDDOS-v2. The tool displayed increased CPU utilization on the target system, demonstrating how flooding attacks consume system resources and degrade service availability.

---

## Defensive Perspective

Continuous performance monitoring enables administrators to identify sudden spikes in CPU, memory, disk, or network usage that may indicate denial-of-service attacks or other abnormal system activity requiring investigation.
