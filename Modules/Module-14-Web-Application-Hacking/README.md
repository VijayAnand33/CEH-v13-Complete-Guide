# Module Summary

This module introduces the methodology used to assess and attack web applications from a penetration tester's perspective. Throughout the module, I learned how to perform web application reconnaissance, spidering, vulnerability scanning, brute-force attacks, Remote Code Execution (RCE), WordPress security assessment, and AI-assisted web application testing using industry-standard security tools.

Rather than focusing only on executing commands, this documentation aims to explain the concepts, attack flow, tools used, and the reasoning behind each lab.
